# Netflix Movies and TV Shows – Exploratory Data Analysis

## Overview
This project performs exploratory data analysis on the Netflix Movies and TV Shows dataset to understand content distribution, trends over time, country-wise availability, and rating patterns.

## Dataset
- Source: Kaggle – Netflix Movies and TV Shows
- Records: ~7,800
- Format: CSV

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Univariate analysis (content type, ratings, countries, release years)
- Bivariate analysis (content trends over time, ratings vs type, country vs type)
- Duration and season analysis

## Key Insights
- Netflix’s catalog is dominated by movies.
- Significant content growth occurred post-2015.
- The United States leads content production.
- Mature-rated content is highly prevalent.
- Most TV shows have fewer seasons.

## Limitations
- No viewership or engagement data available.
- Some missing values required assumptions during cleaning.
- Dataset may not reflect the most recent Netflix catalog.

## Conclusion
This project demonstrates the use of Python for data cleaning, visualization, and exploratory analysis to derive meaningful insights from real-world datasets.
